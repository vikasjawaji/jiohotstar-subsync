Manual Installation Steps

Load the extension codebase locally onto your desktop browser.

1

Clone or Download Files

Download the source repository containing the manifest.json, content.js, and page.js files onto your local hard drive.

2

Enable Developer Mode

Open Google Chrome, navigate straight to chrome://extensions inside your URL bar, and click the "Developer mode" switch in the top-right corner.

3

Load Unpacked

Click the "Load unpacked" button in the top-left area, navigate to your saved extension directory, and select it to install instantly.

Frequently Asked Questions

Quick answers on extension features, performance, and configurations.

Does this affect the overall streaming quality or performance? +
Not at all. SubSync executes direct core timing modifications inside isolated main worlds. It does not monitor internet packets or slow down streaming buffers. It works seamlessly even on low-bandwidth setups.
Why are shortcuts not responding on my screen? +
Make sure that your text cursor is not actively focused inside an input element or text box (such as search boxes). Click anywhere blank on the main player viewport, then hit A or D keys.
How do I input a precise custom offset value? +
Simply double-click directly inside the timer digits of the SubSync sync tab. It turns into an inline text editor. Type your custom offset (e.g., 2.50 or -1.75) and hit Enter to apply instantly.
